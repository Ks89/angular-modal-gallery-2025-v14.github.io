Previews

Previews component contains previews with navigation arrows.


PreviewConfig

Interface to configure previews (not 'side-previews', but those at the bottom) in modal gallery.


visible
Type: boolean

If false, previews will be hidden, otherwise not. true by default.
number
Type: numberoptional

Number of previews to show. 3 by default.
arrows
Type: booleanoptional

If false, preview's navigation arrows are visible, otherwise not. true by default.
clickable
Type: booleanoptional

If false, previous won't be clickable, otherwise not. true by default.
size
Type: Sizeoptional

You can change the default size of previews. width: 50px and height: auto by default.

CarouselPreviewConfig

Interface to configure carousel previews.


visible
Type: boolean

If false, previews will be hidden, otherwise not. true by default.
number
Type: numberoptional

Number of previews to show. 4 by default.
arrows
Type: booleanoptional

If false, preview's navigation arrows are visible, otherwise not. true by default.
clickable
Type: booleanoptional

If false, previous won't be clickable, otherwise not. true by default.
width
Type: stringoptional

You can change the width of previews. 25% by default.
maxHeight
Type: stringoptional

You can change the maximum height of previews ignoring higher values if specified in breakpoints. 200px by default.
breakpoints
Type: BreakpointsConfigoptional

You can change the heights at different window's width. xSmall: 100, small: 100, medium: 150, large: 200, xLarge: 200 by default.

BreakpointsConfig

Interface to customize carousel previews heights at different window's sizes.


xSmall
Type: numberoptional

Number of pixels to be used as height for extra-small screen (defined in @angular/cdk)
small
Type: numberoptional

Number of pixels to be used as height for small screen (defined in @angular/cdk)
medium
Type: numberoptional

Number of pixels to be used as height for medium screen (defined in @angular/cdk)
large
Type: numberoptional

Number of pixels to be used as height for large screen (defined in @angular/cdk)
xLarge
Type: numberoptional

Number of pixels to be used as height for extra-large screen (defined in @angular/cdk)

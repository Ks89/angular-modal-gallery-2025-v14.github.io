Accessibility

Accessibility interface used by all components of this library.

AccessibilityConfig

Interface to configure accessibility features as you want, for instance with a different language.

These are all supported string fields (grouped by components):


  • Background
    • backgroundAriaLabel
    • backgroundTitle
  • Plain gallery
    • plainGalleryContentAriaLabel
    • plainGalleryContentTitle
  • Modal gallery
    • modalGalleryContentAriaLabel
    • modalGalleryContentTitle
  • Loading spinner
    • loadingSpinnerAriaLabel
    • loadingSpinnerTitle
  • Current image
    • mainContainerAriaLabel
    • mainContainerTitle
    • mainPrevImageAriaLabel
    • mainPrevImageTitle
    • mainNextImageAriaLabel
    • mainNextImageTitle
  • Dots
    • dotsContainerAriaLabel
    • dotsContainerTitle
    • dotAriaLabel
  • Previews
    • previewsContainerAriaLabel
    • previewsContainerTitle
    • previewScrollPrevAriaLabel
    • previewScrollPrevTitle
    • previewScrollNextAriaLabel
    • previewScrollNextTitle
  • Carousel
    • carouselContainerAriaLabel
    • carouselContainerTitle
    • carouselPrevImageAriaLabel
    • carouselPrevImageTitle
    • carouselNextImageAriaLabel
    • carouselNextImageTitle
